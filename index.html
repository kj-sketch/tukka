Tukka game - Bano tukkebaaj 😎

Guess the number b/w 1 to maximum number u decided.

#RULES

  • There are total 7 chances in the game because the word "Chances" contains 7 letters and as you know 7 means : Thala for a Reason 😅
  • You can use 2 hints.
  • Type "h1" 👀 to get hint 1. It makes game easier.
  • Type "h2" 👀 to get hint 2. It makes game more easier.
  • Type "quit" to quit the game.
  • Type only INTEGER in the range of 1 to maximum number you selected to get a correct answer.

    • If you are using hint 1 and hint 2 by typing h1 and h2 respectively 👀. then it will be counted as your chance. because - "To win something, u have to lose something ".

    If you are done with reading the rules. Press Right click on your screen interface and select inspect option. after clicking on it. A display appears in which you have to go to Console button Here click Ctrl + l to clear the screen . HERE in CONSOLE screen u will find the output of the game . Now press ctrl+r to Start the game by again setting the difficulty level.
